# **App Name**: PDFZero

## Core Features:

- PDF Manipulation Suite: Enable users to merge, split, compress, and rotate PDF documents seamlessly within the web application.
- Format Conversion: Convert PDF files to various formats, including Word, Excel, PPT, JPG, PNG, and HTML, and vice versa.
- PDF Security Tools: Implement functionalities for password protecting and unlocking PDF files to ensure data privacy.
- OCR Functionality: Integrate Optical Character Recognition (OCR) to convert scanned documents and images into editable and searchable PDF files. A tool will choose the optimal configuration for Tesseract.js.
- PDF Repair: Add ability to repair PDF documents to allow recovery of corrupted data and fix structure of broken files.
- PDF Comparison Tool: Offer a feature to compare two PDF files, highlighting the differences in content, structure, or images. Also try a tool to detect which PDF has better quality and which is missing essential information, so you can save your own data effectively
- Annotation: Support text redaction, highlights, and add text notes

## Style Guidelines:

- Primary color: Saturated cool red (#E44C65), drawing from iLovePDF's styling while maintaining a professional aesthetic.
- Background color: Pale cool red (#F9E7EA) provides a subtle contrast, and aligns with the red primary.
- Accent color: Desaturated blue-purple (#596FE0), to bring visual interest to supporting elements.
- Body and headline font: 'Inter' sans-serif. Note: currently only Google Fonts are supported.
- Crisp, modern icons from 'Lucide Icons' to visually represent different PDF tools.
- Use a clean, grid-based layout to present tools and options clearly, inspired by leading design agencies.
- Subtle Framer Motion animations on user interactions for enhanced UX.
